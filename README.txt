Drupal project page: http://drupal.org/project/postcodeanywhere
GitHub project page: https://github.com/kenorb/postcodeanywhere

DESCRIPTION

  Find your UK address via Postcode.
  This is 3rd party integration with Postcode Anywhere service.

  Note: This is not official module of Postcode Anywhere
  So if you need any other information related to API, please check here:
  http://www.postcodeanywhere.co.uk/support/

INSTALLATION

  1. Install module.

REQUIREMENTs

  You have to be already registered member on PostcodeAnywhere website.
  Site: http://www.postcodeanywhere.co.uk

ALTERNATIVE SOLUTIONs

  https://developers.google.com/maps/articles/phpsqlsearch
  http://www.royalmail.com/marketing-services/address-management-unit/addr...
  http://ben-major.co.uk/2012/02/using-google-maps-to-lookup-uk-postcodes/
